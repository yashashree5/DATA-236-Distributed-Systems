## DATA-236 – Distributed Systems

## Course Description

This course covers the fundamentals and practical implementation of modern distributed systems. Exploring

- Big Data Distributed Systems
- Large-Scale Distributed Systems Frameworks
- Distributed Messaging Frameworks
- Distributed System Development with Cloud Technologies
- REST (Representational State Transfer) Service Development and Deployment

The course focuses on designing reliable, scalable, and fault-tolerant systems.

Assignments and projects may be added throughout the semester.
